# Privacy Policy Draft (Merchant-Facing)

## Purpose
This app processes personal data only to provide WhatsApp CRM and Shopify order workflow features for merchants.

## Data We Process
- Customer identifiers: name, email, phone.
- Order-related data: order id, order amount, order status, shipping address fields.
- Conversation and message metadata required for support and automation.

## Why We Process Data
- Customer support and conversation management.
- Order confirmation/cancellation workflows via WhatsApp.
- Shopify to CRM synchronization and order lifecycle tracking.
- Operational analytics and service reliability.

## Data Minimization
We process only data required for the above functions and do not use customer data for unrelated purposes.

## Data Sharing
We do not sell customer data. Data is shared only with subprocessors required to run the service (for example, infrastructure, messaging providers, Shopify API).

## Retention
Data is retained only as long as required for service delivery, legal obligations, and dispute handling, then deleted or anonymized per retention policy.

## Security
- Encryption in transit (HTTPS/TLS).
- Role-based access controls.
- Authentication and authorization controls.
- Audit events for critical operational actions.

## Merchant Rights
Merchants can request data export/deletion and can disconnect integrations.
